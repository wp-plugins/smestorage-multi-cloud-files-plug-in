== Description ==
Back your WordPress blog up to any one of 10 storage clouds including S3, CloudFiles, MobileME, Gmail & Box.net

== Installation ==
Please note the following with this Plug-In

1. Copy the directory to the WordPress Plug-Ins Directory. The directory name needs to "wp-db-backup"
2. Login to WP-Admin and activate the plug-in
3. If you get a message about creating a Backup directory in the WP-Content directory, please create this directory and give it CHMOD 777 permissions
4. Ensure you have a valid smestorage.com account and have setup at least one cloud provider where you would like the backup to reside.
5. Enter your smestorage username and details
6. Ensure the backup option is set to backup to smestorage
7. Click the backup button
8. Your WordPress DB will now be backed up to SMEStorage and will reside in the "My WordPress Backups" directory.
